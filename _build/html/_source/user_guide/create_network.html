Create Network and Subnet¶

  1. Login to Openstack Dashboard as normal user (demo).

  2. Navigate to Network. Select Network Topology or Networks.

  3. Click on “Create Network”. Enter the Network name and set Admin State UP. Click next

  4. Set the options Subnet Name, Network Address(CIDR Format), IP Version and Gateway IP. Click next

  5. Set additional details as per your requirement or leave blank. Click create.

  6. Once you have created Network, Network topology will look like below.

Create router¶

  1. Click on “Create router”. Enter Router Name, set Admin State UP and External Network VJTINet.

  2. Navigate to Routers. Click on the router you just created.

  3. Select the “Interfaces” tab and click on “Add Interface”.

  4. Select “Subnet”, IP Address for router interface(Optional). Click on Add Interface.
