Installation of Ubuntu Server 14.04
===================================

1. Boot from disk or usb
2. Select language
3. Install Ubuntu server
4. Choose the language for the installation process
5. Select your location. E.g. Country Name
6. Configure the keyboard
7. Country of origin for the keyboard
8. Select keyboard layout
9. Configure the network
10. Set hostname
11. Set up users and passwords
12. Encrypt home directory or leave
13. Configure the clock
14. Partition disk
15. Unmount partitions that are in use
16. Do the partitioning as per your convenience
17. Select / <root> partition and setup swap area
18. Write changes to disk
19. Configuring tasksel e.g. update settings
20. Software selection
21. Install the GRUB boot loader on a hard disk
22. Complete!!
