Installation Guide..
====================


To deploy VJTICloud following sources are used as references for configuring various services.

#. `OpenStack Installation Guide for Ubuntu 14.04 <http://docs.openstack.org/kilo/install-guide/install/apt/content/>`__
#. `A Beginner's Guide To LVM <https://www.howtoforge.com/linux_lvm>`__
#. `Ubuntu LVM <https://wiki.ubuntu.com/Lvm>`__
#. `LVM wiki <https://en.wikipedia.org/wiki/Logical_Volume_Manager_(Linux)>`__
#. `Set up NAT with Linux and iptables <http://www.bctes.com/nat-linux-iptables.html>`__
#. `Add an additional IP address to an interface in Ubuntu 14 <http://askubuntu.com/questions/585468/how-do-i-add-an-additional-ip-address-to-an-interface-in-ubuntu-14>`__
#. `Configuring virtual network interfaces in Linux <https://linuxconfig.org/configuring-virtual-network-interfaces-in-linux>`__
#. `How to disable User Accounts on Linux System <https://linuxconfig.org/how-to-disable-user-accounts-in-linux>`__
#. `Multiple external networks with a single L3 agent <http://blog.oddbit.com/2014/05/28/multiple-external-networks-wit/>`__
#. `Load Balancer as a service <https://access.redhat.com/documentation/en/red-hat-enterprise-linux-openstack-platform/7/networking-guide/chapter-12-configure-load-balancing-as-a-service-lbaas>`__
#. `Firewall as a service <https://access.redhat.com/documentation/en/red-hat-enterprise-linux-openstack-platform/7/networking-guide/chapter-16-configure-firewall-as-a-service-fwaas>`__
